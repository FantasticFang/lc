#### KMP算法

- KMP算法的核心是，求解某一个字符串相等真前缀、真后缀的最大长度（真前缀/真后缀 != 字符串本身）。对于字符串s="abcasabc"
  - 前缀函数：f(0) = 0， 因为"a"没有自己的真前缀和真后缀
  - f(1) = 0
  - f(2) = 0
  - f(3) = 1，因为s[0] = s[3]
  - f(4) = 0
  - f(5) = 1，因为s[0] = s[5]
  - f(6) = 2，因为s[0, 1] = s[5, 6]
  - f(7) = 3，因为s[0, 2] = s[5, 7]
- 前缀函数的性质，f(n) = j，那么s[0, j-1] = s[n-j+1,n]
